This widget adds a Sketchfab profile widget in your WordPress website Sidebar or Footer area if your theme supports that. There is also a short code for POST and PAGE.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

For more details,

http://codex.wordpress.org/Managing_Plugins
